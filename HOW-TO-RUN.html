How to run the CALFIRE Form Populator application

To run the program...

  • Download the entire package "cdf_autopop".
  • Open the computer terminal and navigate to the directory that contains the folder "cdf_autopop"
  • Click into the "src" folder. Then type the following javac */*.java to compile the program.
  • Execute the program using java controller/APController while stil inside the "src" folder.
